# Kindred Healthcare (kindred-healthcare)
Kindred Healthcare operated a nationwide network of long-term acute care hospitals and inpatient rehabilitation facilities. The Kindred name was retired after the 2021 merger with LifePoint Health, with the acute care facilities operating under the ScionHealth brand and rehabilitation facilities continuing as Kindred Hospital Rehabilitation Services. No public REST APIs or developer portal have been identified for the legacy Kindred Healthcare or its successor entities.
